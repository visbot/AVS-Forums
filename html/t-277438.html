Super Nova Preset
This Preset use the rotate polar rose function but in different way then below.

Now it's more like a rotated String then a polar rose.

This polar rose function is the first one that a used to made presets.
Actually, I am fascinated by it, because you can rotate it, use it like a string, wobble it and use it like a rotated infinity symbol.

Use a simple preset with only a superscope and try this.

Rotated Infinity String

init: n=9000;theta=0.01;a=0.6;k=64;
frame: theta=theta+$PI/8;
point: r=a*cos(k*v-theta);x=r*cos(v);y=r*sin(k*v);


Wobbling String

init: n=9000;theta=0.01;a=0.6;k=64;
frame: theta=theta+$PI/8;
point: r=a*cos(k*v);x=r*cos(k*v+theta);y=r*sin(k*v);


Spider String/Works only for the super nova preset

init: n=9000;theta=0.01;a=1.3;k=8;
frame: theta=theta-$PI/32;
point: r=a*cos(k*v+theta);x=r*cos(v+theta);y=r*sin(k*v);


You'll enjoy it.
I promise you.
